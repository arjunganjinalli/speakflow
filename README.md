# SpeakFlow

AI Dialogue Practice Coach powered by GLM 5.1

## What it does

Practice dialogue in 11 languages with an AI coach. Two modes:
- **Practice Mode**: AI evaluates your spoken/typed responses for accuracy, grammar, and fluency
- **Presentation Mode**: AI speaks all other lines; you practice saying your lines in real-time

## Features

- Multi-language support (English, Spanish, French, German, Chinese, Japanese, Korean, Italian, Portuguese, Russian, Arabic)
- Web Speech API for speech recognition & text-to-speech
- GLM 5.1 API for intelligent dialogue evaluation
- Real-time feedback with score, accuracy, grammar, fluency analysis
- Beautiful dark UI with glass morphism design
- Session tracking and completion stats

## How to Use

1. Open `SpeakFlow.html` in a web browser
2. Paste your GLM 5.1 API key (get from z.ai)
3. Choose language, paste your dialogue script
4. Select your role
5. Start practicing!

## Built with

- HTML5, CSS3, JavaScript (ES6+)
- Web Speech API (recognition & synthesis)
- GLM 5.1 API via z.ai
- Tailwind CSS + custom styling
